HTML Element Purpose Common Attributes Attribute Purposes
<table> Contains all the tabular data None N/A
<tr> Defines a row in the table None N/A
<td> Defines a cell within a table row colspan, rowspan colspan: Number of columns to span across
rowspan: Number of rows to span across
<th> Defines a header cell (title) within a table row scope scope="col": Specifies header for a column
scope="row": Specifies header for a row
<thead> Sections off the table's header None N/A
<tbody> Sections off the body of the table None N/A
<tfoot> Sections off the bottom part of the table None N/A
